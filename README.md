# Smart Budget & Expense Manager

A full-stack app for Indian families to track expenses, manage bills, and build financial discipline.

## Tech Stack
- Frontend: React.js, HTML, CSS
- Backend: Flask (Python)
- Database: PostgreSQL
- Auth: JWT
- Deployment: Docker, Cloud
- CI/CD: GitHub Actions

## Features
- User authentication & profiles
- Daily expense tracker
- Budget dashboard (charts)
- Bill/EMI reminders
- Family mode (shared budgets)
- Regional language support
- AI smart suggestions

---

## Structure
- `backend/` - Flask API
- `frontend/` - React app
- `db/` - SQL scripts
- `docker/` - Dockerfiles
- `.github/` - CI/CD workflows
